# Truesofts

This is the official repository for the Truesofts project.

## How to use

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/Truesofts.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
